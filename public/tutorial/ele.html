如何贡献饿了么 Cookie

  1. PC 登录微信，并给自己分享一个饿了么外卖红包链接（领过的也行）

  2. 浏览器设置：1. 建议使用chrome浏览器。2. 如果要贡献多个cookie，可以开启无痕模式。
    按 F12 打开开发者工具，点击下图中的按钮，切换成 mobile 调试模式。

  3. 点击当前模拟的设备名称，在下拉框中选择"Edit"。

  4. 点击"Add custom device..."。

  5. 输入 device name ( 随便写一个 ) 和 User agent string ( Mozilla/5.0 (Linux; Android 5.1; m1 metal Build/LMY47I; wv) AppleWebKit/537.36 (KHTML, like Gecko) Version/4.0 Chrome/53.0.2785.49 Mobile MQQBrowser/6.2 TBS/043409 Safari/537.36 V1ANDSQ7.2.5744YYBD QQ/7.2.5.3305 NetType/WIFI WebP/0.3.0 Pixel/1080 )，点击"Add"

  6. 点击步骤3中的设备名称，切换成刚才添加的设备，然后刷新页面。

  7. 填写 qq 帐号，登录。然后在开发者工具切换tab至Console,输入document.cookie,回车，复制打印的信息。

  8. 将上一步复制的 Cookie 粘贴到输入框，点击确定即可。

  9. 如在操作中碰到问题，可以扫下面二维码，联系管理员。
